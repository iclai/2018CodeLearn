# Column欄

在Bootstrap中，&lt;div&gt;區塊欄位，會按照父類別的寬度大小來做改變，套用col會平均分配欄寬。

```markup
<div class="container">
        <div class="row">
            <div class="col bg-primary">ROEW1</div>
            <div class="col bg-danger">ROEW2</div>
            <div class="col bg-warning">ROEW3</div>
        </div>
    </div>
```
